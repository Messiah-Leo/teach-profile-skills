# Routing

Use this reference when deciding how `teach` and `learner-profile-teach` combine.

## Teaching Intents

Route through `learner-profile-teach` before `teach` for:

- teach me
- help me learn
- next lesson
- continue the lesson
- course design
- research derivation as a lesson
- 教我
- 教会我
- 带我学
- 学一下
- 讲一下
- 给我上一课
- 继续上一课
- 帮我理解

## Non-Teaching Tasks

For narrow coding, file, GitHub, or system tasks, do not force the teaching
workflow unless the user asks to learn, reflect, or record durable course notes.

## Conflict Rule

Current user instructions override the stable profile. If the user asks not to
record in Obsidian, do not mirror lesson notes unless required by a separate,
explicit active workflow.

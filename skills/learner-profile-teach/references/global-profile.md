# Global Learner Profile

Customize this file locally. Do not publish private details.

## Identity

Describe the learner's stable role or learning identity.

Examples:

- researcher,
- engineer,
- designer,
- student,
- manager,
- independent learner.

## Long-Term Learning Goals

Record durable goals that should shape many learning sessions.

Examples:

- improve research-method reasoning,
- build practical engineering judgment,
- make better decisions,
- learn a new field deeply,
- improve planning and self-regulated learning.

## Core Workstream

Describe the learner's recurring work or study loop.

Template:

```text
context/constraints -> model/plan -> method/action -> implementation/practice
-> feedback/evaluation -> revision
```

## Stable Teaching Orientation

Core concepts, professional terms, definitions, and distinctions matter. Do not skip them.

Avoid empty abstraction. Teach concepts inside usable contexts with examples, counterexamples, and transfer tasks.

## Highest-Value Transfer Contexts

List the contexts that help the learner understand and apply abstract ideas.

Examples:

- research work,
- software engineering,
- writing,
- management,
- planning,
- personal cognition,
- creative practice.

## Priority Rule

Current user messages override this profile. Treat the profile as a collaboration model, not a fixed label.

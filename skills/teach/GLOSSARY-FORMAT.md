# GLOSSARY.md Format

`GLOSSARY.md` is the canonical vocabulary for the teaching workspace.

## Structure

```md
# {Topic} Glossary

One or two sentences describing the glossary scope.

## Terms

**Term**:
Concise definition.
_Avoid_: ambiguous alternatives if needed.
```

## Rules

- Add a term only when the user can use it correctly.
- Keep definitions tight.
- Pick one canonical term when aliases exist.
- Revise definitions as understanding deepens.
- Group terms under headings when clusters emerge.

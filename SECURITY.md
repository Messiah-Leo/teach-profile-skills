# Security And Privacy

This repository contains skill instructions and templates. It should not contain
private learner data.

## Do Not Commit

- Personal learner profiles with sensitive details.
- Private research notes or client information.
- Local note-vault paths.
- API keys, access tokens, cookies, or credentials.
- Machine-specific environment files.

## Reporting

If you find a privacy or security issue, open a GitHub issue with the smallest
safe reproduction. Do not include secrets or personal data in the issue.

For urgent private reports, contact the repository owner through their preferred
GitHub profile contact method.

## Maintainer Checklist

Before merging changes, check:

- no private profile details are included,
- examples are generic,
- documentation does not instruct silent memory updates,
- installer scripts do not overwrite unrelated user files,
- `.gitignore` still excludes local learning workspaces and generated outputs.
